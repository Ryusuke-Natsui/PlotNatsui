# 機能仕様メモ

## 現在入れてある機能
### 入力
- 複数ファイル読み込み
- txt / csv / tsv の自動判定
- 最初の数値 2 列を x, y として採用

### 表示
- 重ね描き
- 凡例
- 縦方向オフセット
- テーマ切替

### ピーク検出
- 簡易 local maxima 探索
- minimum prominence
- minimum distance

### 正規化
- 選択したピークの高さが 1 になるように全体を割る

### エクスポート
- PNG
- 設定 JSON
